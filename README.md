xgame框架

## 功能
该框架是有TypeScript语言开发，方便游戏开发，主要提供了一下功能：
```
  1. 状态机
  2. 游戏管理器
    a. AudioMgr 游戏音乐管理器
    b. DataMgr 数据管理器
    c. NetworkMgr 网络管理器
    d. PlatformMgr 平台管理器, 兼容各种小游戏平台
    e. StaticDataMgr 静态数据管理器，导出excel数据  
    f. SystemMgr 系统数据
    e. UIAssetsMgr UI资源管理器
    g. UIMgr UI管理器
```

## 注意事项
```
 1. 编译依赖CocosCreator2.x版本，需要自行下载
```
